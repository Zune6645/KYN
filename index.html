Introduction

This is a portal that provides APIs for user register, login and also add, update, delete stores and even search stores. Provides Google Sign in API.

This project is written for the purpose of fulfilling one of the BTEC assignments. We were instructed to create a portal while implementing Google API.

  • REST
  • React
